## 1. 整体架构

```
┌──────────────────────────────────────────────────────────────┐
│                    Electron 桌面应用                          │
│                                                              │
│  ┌─────────────┐    IPC     ┌──────────────────┐            │
│  │ 渲染进程     │ ◄═══════► │    主进程          │            │
│  │ Vue 3 + TS   │           │    Electron main   │            │
│  │ Tailwind CSS │           │                    │            │
│  └─────────────┘           └────────┬─────────┘            │
│                                      │ spawn                 │
│                                      ▼                       │
│                            ┌──────────────────┐            │
│                            │  Python 后端       │            │
│                            │  Flask + Crawler   │            │
│                            │  (Playwright)      │            │
│                            └──────────────────┘            │
└──────────────────────────────────────────────────────────────┘
```

### 通信方式

| 链路 | 协议 | 说明 |
|------|------|------|
| 渲染进程 ↔ 主进程 | Electron IPC | `ipcRenderer.invoke` / `ipcMain.handle` + `contextBridge` |
| 主进程 → Python 后端 | HTTP REST | `http://127.0.0.1:15678/api/*` |
| 主进程 → 渲染进程 | IPC 事件推送 | `webContents.send('download:progress')` |

| **Electron** | 30.0.1 | 桌面应用框架 |

### 主要职责

1. **窗口管理**：创建 BrowserWindow、无边框窗口、自定义标题栏
2. **Python 进程管理**：spawn Python Flask 服务、监控进程状态、优雅退出
3. **IPC 路由**：注册所有 `ipcMain.handle`，转发请求到 Python HTTP API
4. **进度轮询**：下载任务启动后，1秒间隔轮询 `/api/progress/{taskId}`，推送到渲染进程
5. **图片代理**：`net.fetch` 代理下载跨域图片，解决 Referer 限制

## 7. 关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 浏览器自动化 | Playwright（统一 missav + kissjav） | 跨平台、API 丰富、维护活跃、统一技术栈 |
| HTTP 客户端 | httpx > requests > aiohttp | 原生 async、HTTP/2、流式下载 |
| 视频合并 | ffmpeg m3u8 合并 > 手动拼接 | 成熟稳定、支持所有编码格式 |
| 进程通信 | HTTP REST > WebSocket > stdio | 简单可靠、易于调试 |
| 进度推送 | 轮询 > WebSocket | 实现简单、1秒延迟可接受 |
| CSS 方案 | Tailwind CSS > Scoped CSS > CSS-in-JS | 开发效率高、体积小 |
