# MissAV 视频下载器 - Windows 11 原生应用开发手册

## 📋 项目概述

### 项目目标
将 F:\novel 项目中的 missav 视频爬虫封装为 Windows 11 原生桌面应用程序，参考 [videdown](https://github.com/cshuangyy/videdown) 的技术栈和 UI 设计。

### 核心功能
1. **视频解析** - 从 missav.ws 解析视频信息（标题、封面、演员、标签等）
2. **视频下载** - 下载 m3u8 流媒体视频并合并为 mp4
3. **下载管理** - 下载队列、进度显示、暂停/恢复
4. **历史记录** - 已下载视频的浏览历史

        async with async_playwright() as p:

### 2. Electron 主进程（IPC 通信）

```typescript
// electron/main.ts
import { app, BrowserWindow, ipcMain, dialog, shell } from 'electron'
import { spawn } from 'child_process'
import path from 'path'

// Python 后端进程
let pythonProcess: ChildProcess | null = null

// 启动 Python 后端
function startPythonBackend() {
  const pythonPath = path.join(__dirname, '..', 'python', 'server.py')
  pythonProcess = spawn('python', [pythonPath])

  pythonProcess.stdout?.on('data', (data) => {
    console.log(`Python: ${data}`)
  })

  pythonProcess.stderr?.on('data', (data) => {
    console.error(`Python Error: ${data}`)
  })
}

// IPC 处理器
ipcMain.handle('video:parse', async (event, url: string) => {
  // 调用 Python 后端解析视频
  const response = await fetch('http://localhost:5000/parse', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url })
  })
  return response.json()
})

ipcMain.handle('video:download', async (event, options) => {
  // 调用 Python 后端下载视频
  const response = await fetch('http://localhost:5000/download', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(options)
  })
  return response.json()
})

ipcMain.handle('download:progress', async (event, taskId: string) => {
  // 获取下载进度
  const response = await fetch(`http://localhost:5000/progress/${taskId}`)
  return response.json()
})
```

      </div>
    </div>

<script setup lang="ts">
import { ref } from 'vue'

const url = ref('')
const isParsing = ref(false)
const videoInfo = ref(null)
const downloadQueue = ref([])

async function parseVideo() {
  isParsing.value = true
  try {
    const result = await window.electronAPI.video.parse(url.value)
    videoInfo.value = result
  } finally {
    isParsing.value = false
  }
}

async function startDownload() {
  const task = await window.electronAPI.video.download({
    url: url.value,
    outputDir: settings.value.downloadDir
  })
  downloadQueue.value.push(task)
}
</script>

## ⚠️ 注意事项

1. **法律风险**: 请确保仅用于个人学习和研究目的
2. **网站变化**: missav.ws 可能更新页面结构，需要定期维护爬虫
3. **网络环境**: 可能需要代理才能访问目标网站
4. **资源占用**: 视频下载会占用大量带宽和磁盘空间
5. **版本管理**: 每次更新代码重新打包都必须更新 `package.json` 中的版本号

### 版本号规则

```json
// package.json
{
  "version": "1.0.0"  // 主版本.次版本.修订号
}
```

- **主版本 (major)**: 重大功能变更或不兼容更新
- **次版本 (minor)**: 新增功能，向后兼容
- **修订号 (patch)**: Bug 修复，小改动

打包后安装程序自动包含版本号: `MissAV Downloader Setup x.x.x.exe`
